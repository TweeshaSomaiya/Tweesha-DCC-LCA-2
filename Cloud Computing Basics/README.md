# Cloud Computing Deployment

## Aim

To deploy a simple application using cloud computing.

## Objective

* To understand cloud deployment
* To host an application on cloud
* To access it through a browser

## Description

In this experiment, a simple Flask web application is deployed on a cloud platform. The application is hosted online and accessed using a public URL.

## Steps

1. Created a Flask application
2. Added required dependencies
3. Uploaded project to GitHub
4. Connected repository to cloud platform
5. Deployed the application
6. Accessed it via browser

## Result

The application was successfully deployed and accessed through a browser.

## Conclusion

Cloud platforms allow applications to be deployed and accessed globally without local setup.
